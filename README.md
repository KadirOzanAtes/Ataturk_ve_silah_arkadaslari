NASIL OYNANIR?
Mouse sol tuş ile asker seçebilir ve sağ tuş ile de askeri yürütebilirsiniz.
Çoklu asker seçmek için mouse sol tuş ile basılı tutarak seçmek istediğiniz askerlerin üzerinde alan çizebilirsiniz.
Mavi renkli düşman askerlerin yakınına türk askeri götürdüğünüz zaman otomatik olarak savaşmaktadır.
